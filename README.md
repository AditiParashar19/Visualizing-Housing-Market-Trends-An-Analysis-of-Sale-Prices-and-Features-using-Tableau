# Visualizing-Housing-Market-Trends-An-Analysis-of-Sale-Prices-and-Features-using-Tableau

## Project Overview

This project focuses on analyzing housing market data and visualizing key trends using Tableau. The objective is to explore how different property features influence house sale prices and to identify patterns in the housing market.

Through interactive dashboards and visualizations, the project provides insights into factors affecting housing prices such as location, property size, number of rooms, and other attributes.

## Objectives

- Analyze housing market data to understand price trends.
- Identify key factors that influence house sale prices.
- Create interactive visualizations using Tableau.
- Provide insights that help in data-driven decision-making

## Tools and Technologies

- Tableau – Data visualization and dashboard creation
- Excel / CSV Dataset – Data source
- Data Cleaning & Transformation – Tableau Public
